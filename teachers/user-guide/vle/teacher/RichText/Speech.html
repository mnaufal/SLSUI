Speech Evalution

Enable Speech Evaluation

  1. Teachers can enable auto-marking in an Audio-Response Question by selecting the checkbox for auto-marking with the relevant Speech Evaluation engine in the dropdown.

    Note: The Speech Evaluation feature is only available for Audio-Response Question types.

  2. After auto-marking has been selected, teachers will be able to click the Speech Evaluation icon in the Rich Text editor.


  3. In the pop-up that appears, add the text for evaluation and click Save.

  4. After saving the text to be evaluated using the Speech Evaluation tool, teachers may continue to edit the question. Teachers may also edit the pre-populated rubrics by customising the bands and mark range.

  5. If there are no further changes to the question, click Save.


Speech Evaluation Teacher Feedback

  1. In a Teacher-Marked Quiz, teachers can edit the suggested marks returned by the Speech Evaluation tool, if needed.

  2. Teachers can also add personalised feedback to help students understand their areas for improvement.
