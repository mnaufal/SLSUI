About Learning Feedback Assistant (LFA)(New)

Note: Learning Feedback Assistance (LFA) is currently undergoing pilot in participating schools and will be available for wider access upon conclusion of the pilot.

  1. Learning Feedback Assistant (LFA) - English Language (EL) is only available for English Language teachers. It is a tool that can return feedback on the incorrect use of grammar rules for each student. Teachers may edit the suggested feedback returned by the tool and add additional feedback for students to help them better understand the areas for improvement.

  2. In a Teacher-Marked Quiz, in the Free Response Question, check the box to enable Language Feedback (EL).

  3. Enabling LFA-EL will only allow plain text input.

  4. Save the component and return to lesson editor mode. You should see a prompt that LFA-EL has been enabled for the question.

  5. When the quiz is submitted by the student, teachers will be able to see the errors underlined as suggested by LFA-EL.

  6. Teachers will be able to see the Language Feedback card anchored to the response, and add comments in the card to provide further feedback. The teacher may also select ‘Edit’ to amend the suggested feedback.


Note:

  • When uploading Question Item XML into an FRQ, only image assets in the Question Body will be processed, and the processing will fail if the value provided in Question Item XML exceeds SLS’ maximum limits (e.g. maximum character limit for Question Body).
