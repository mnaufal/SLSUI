View and Edit a Lesson/Course Plan (New)

The yellow navigation arrow in the Lesson/Course Plan indicates your current view of the Lesson/Course.

View a Lesson/Course Plan

  1. In the Lesson/Course View Mode, click the Course Plan icon in the collapsed Activity List. All the Sections/Activities in a Lesson/Course will be shown.

  2. The yellow navigation arrow in the Lesson/Course Plan indicates your current view in the Lesson/Course.

  3. For a Course, clicking the Section from the Course Plan will bring you to the particular Section.

  4. You can copy a Lesson/Course by clicking the Overflow (3-dot icon) and selectingÂ Make a Copy.

  5. To view the list of Activities within a Section, click Show Activities ( ). The number indicated in the bracket shows the number of Activities within the Section.

  6. Clicking the Activity from the Course Plan will bring you to the particular activity.


Edit a Lesson/Course Plan

  1. In the Lesson/Course View Mode, click the Pencil icon to edit a Course.

  2. Click the Lesson/Course Plan icon to open the Course Plan.

  3. To edit the titles of a Lesson/Course/Section/Activity, click the Pencil icon beside each field.

  4. Click Save to confirm.

  5. Click the Overflow(3-dot icon) to Add Section/Activity before or after, Move Section/Activity Up/Down, Delete a Section and/or an Activity. For Activities, you can also Duplicate Activity and select Set as Optional.

    Note: You can reorder activities to a different section by using the Move up/down feature.

  6. Alternatively, you can click Add Section, found below the last Section of the Course, or Add Activity, found below the last Activity of each Section/Lesson.

  7. Select the type of Activity you wish to add from Add New Activity.

  8. Click X to exit the Lesson/Course Plan.


Section Prerequisites (For Courses Only)

You can add Section Prerequisite to a course. Sections indicated with Prerequisites will be locked until students complete the required section.

  1. In the Course Editor, click the Course Plan icon.

  2. To add Progressive Completion Prerequisites, select the checkbox next to "Sections can only be accessed when the previous section is completed." and click Apply.

    Note: This ensures each Section can only be accessed when the Section before is completed.

  3. To add Custom Prerequisites, deselect the checkbox next to "Sections can only be accessed when the previous section is completed." and click Apply.

  4. To add Prerequisites to selected Sections, click Add Prerequisite and check Section(s) students need to complete before accessing the selected Section from the dropdown list. You can check more than 1 Section.

  5. Click Ok to proceed.

  6. The number of Prerequisite Sections students need to complete is indicated in parenthesis. Click Prerequisites ( ) to view the section(s) that require completion.

  7. To make edits, click the Pencil icon, followed by Ok to confirm.
