Add a Free-Response Question

Add Free-Response Question

  1. In the Lesson or Course Editor, click Add Component to expand the toolbar.

  2. In the toolbar, click Free-Response.

  3. You may insert an existing Question from Library or Starred Resources by selecting Add Existing from the toolbar.


Edit Free-Response Question

  1. In the Free-Response Question editor, add the following details:

    • Recommended Time (optional) - You can set the recommended time for the student to complete the question.

    • Use Language Feedback (EL) to provide Marks and/or Feedback (optional).

    • Question Body - Type your question here.

    • Allow file submission checkbox (optional) - Check to turn the Free-Response Question into a File Submission question. If checked, students have the option of uploading a file when they submit their responses. Students can upload up to 10 files. File limits are based on the file type.

    • Pre-populated Student Answer (optional) -

      • This allows teachers to insert an answer stem to scaffold students' responses, or to provide an image for students to work on. When students answer the question, they will be able to directly edit this answer stem or image. Students can edit the images provided by clicking the Edit Drawing icon and drawing on it.

      • (New in R18) Apart from text and media, teachers can also pre-populate files from their Google Drives. When doing so, a copy of the file will be created for individual students to work on. Teachers can directly mark on those files and return them back to students.

    • Add Rubrics (optional) - This allows students to view the rubrics while they attempt the question.

    • Suggested Answer (optional) - The suggested answer to the question. This is shown to students after they have submitted their answers.

    • Feedback (optional) - The explanation to the question's answer(s). This is shown to students after they have submitted their answers.

    • Recommended Time (optional) - You can set the recommended time for the student to complete the question.

    • Marks (optional) - You can set marks for the question.

      Note: Marks will have to be awarded before students can view the Suggested Answer and Feedback.

    • Show and use rubric marks checkbox (if Rubrics are used) - You can award marks using Rubrics for the question.

  2. Click Save after you are done editing. You can then add Keyword/Question Tags by clicking Edit under the Free-Response Details Card. You can view/hide the Free Response Details Card by clicking the Settings icon.

    • Include in Learning Progress checkbox (optional) - You may choose to include this question in the Learning Progress. Click here to find out more about Learning Progress.

      Note: Free-Response and Audio-Response Questions can be marked and included on the Learning Progress only in a Teacher-Marked Quiz.

    • Keyword Tags (optional) - This will improve the searchability of the question. You will have to tag a question if you wish to include it in the Learning Progress.

    • Question Tags (optional) - You may choose to tag individual questions to the relevant topics by clicking  Add Question Tag.

    • Copy Tags from Lesson - This is to tag the question according to the Lesson Tag (if any).
