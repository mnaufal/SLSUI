Add Conditions for Game Stories and Achievements (New)

You can add conditions to Game Stories and Achievements in a gamified Course. Game Stories and Achievements will appear after all Conditions have been fulfilled. Game Stories and Achievements with no Conditions will not appear but can be awarded to students manually.

Conditions that can be added are included in the table below:

Conditions Game Story/Achievement will appear
Section Start students click Enter on the Section cover page
Section Completion shortly after students arrive on the Section end page
Awarded Marks for Quiz shortly after students complete the quiz
Game Story after another Game Story appears
Level after students reach the level
Achievement after another Achievement appears
(applies to Achievements only - Game Story cannot trigger
Achievements)

Note: Section Start Condition cannot be added with other Conditions.

  1. To add Conditions to Game Stories/Achievements, you need to have an existing Game Story/Achievement. Click here to find out how to create a Game Story/Achievement.

  2. The number of Conditions is indicated in parenthesis.

  3. You can find out how to add to Game Stories and Achievements for the scenarios below:

    1. In the Game Story/Achievement Card, click Add Conditions.

    2. In the Add Conditions subpage, click Add Conditions and select Section Start.

    3. From the left dropdown menu, choose a Section within your Course.

    4. Click Save to proceed.

    5. The Game Story/Achievement will appear when students click Enter on the Section Cover of the Assignment.

    Note: Section Start condition cannot be added with other conditions.

    1. In the Game Story/Achievement Card, click Add Conditions.

    2. In the Add Conditions subpage, click Add Conditions and select Section Completion.

    3. From the left dropdown menu, choose a Section within your Course.

    4. Click Save to proceed.

    5. The Game Story/Achievement will appear shortly after students arrive on the Section End page.

    1. In the Game Story/Achievement Card, click Add Conditions.

    2. In the Add Conditions subpage, click Add Conditions and select Awarded Marks for Quiz.

    3. From the left dropdown menu, choose a Quiz Activity.

    4. Enter in the minimum percentage required to trigger the Game Story/Achievement.

    5. Click Save to proceed.

    6. The Game Story/Achievement will appear shortly after students complete the quiz.

    Note: For Teacher-Marked Quiz, Game Story/Achievement will appear after the Quiz is released to students.

    1. In the Game Story/Achievement Card, click Add Conditions.

    2. In the Add Conditions subpage, click Add Conditions and select Game Story.

    3. From the left dropdown menu, choose a Game Story.

    4. Click Save to proceed.

    5. The Game Story/Achievement will appear after the Game Story appears.

    1. In the Game Story/Achievement Card, click Add Conditions.

    2. In the Add Conditions subpage, click Add Conditions and select Level.

    3. Enter the Level required to trigger the Game Story/Achievement.

    4. Click Save to proceed.

    5. The Game Story/Achievement will appear after students reach the level.

    1. In the Achievement Card, click Add Conditions.

    2. In the Add Conditions subpage, click Add Conditions and select Achievement.

    3. From the left dropdown menu, choose an Achievement.

    4. Click Save to proceed.

    5. The Achievements will appear one after another.

    1. Do not set any Conditions to the Game Story/Achievement. Click here to find out more.

    2. In the Add Conditions subpage, click Add Conditions and select Achievement.

    3. From the left dropdown menu, choose an Achievement.

    4. Click Save to proceed.

    5. The Achievements will appear one after another.

  4. To edit Conditions, click the Pencil icon beside Conditions ( ). Click Save to confirm the changes.

  5. To delete a Condition, click the Pencil icon beside Conditions ( ), followed by the Delete icon. Click Save to confirm the changes.
