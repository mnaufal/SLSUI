Add Activity using Activity Templates

Note: Starting from R18, activities (including Quizzes) have the option of being optional. This can be done by checking the box beside Set Activity/Quiz as optional. Optional Activities/Quizzes are indicated with in the Activity List.

Teachers can create Team Activities in Quizzes for Free-Response Questions and Audio-Response Questions.

  1. In the Lesson or Course Editor, there are multiple ways to add an Activity:

    • Click the Expand Activity List icon to open the Full Activity List. Select + Activity.

    • Click the Expand Activity List icon to open the Full Activity List. Click the Overflow (3-dot icon) found to the right of an Activity and click Add Activity After/Before.

    • From the collapsed Activity List, click + and select Activity.

    • From the Lesson/Course Plan, click Add Activity.

  2. From the Add New Activity subpage, scroll the page to view all the Activity Templates available.

  3. Alternatively, you can select an Active Learning Process (or Others) from the dropdown list to filter for specific Activity Templates.

    Note: Activity Templates will not be found under Custom

  4. From the list of Activity Templates in the category, click Choose on the Activity Template Card to view details.

  5. On the Activity Template subpage, you can view details of the Activity Template and select the Activities you wish to add to your Lesson or Course from the Activity List. To clear the selected list of Activities, click Clear.

  6. Click Choose to add the Activities into your Lesson or Course.

  7. The chosen Activity Template has been tagged to an Active Learning Process. To view and edit the Activity Notes and Active Learning Process in the Activity Details Card, click Edit on the Activity Details Card or Edit Details from the Overflow (3-dot icon). These fields are optional and will not be visible to students.
