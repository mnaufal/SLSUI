Manage Gamification Settings (New)

You can only turn on Gamification in a Course Editor.

Turn on Gamification

  1. Open the left menu and click Resources followed by Owned by Me under My Drive.

  2. Select the Course you wish to introduce Gamification to. Click the Pencil icon to enter Course Editor.

  3. In the Course Editor, open the right menu and click Gamification to open the Gamification subpage.

  4. Click the toggle on the top-right corner of the subpage to turn on Gamification.

  5. In the Details tab, fill in the following details:

    • Title - The game title will be pre-filled for you as “Untitled Game”. Change the Title to one that is concise and student-friendly.

    • Description - This description is visible to students in the Game Detail subpage when the Course is assigned. You can include details such as game instructions and story synopsis.

    • Featured Images - These images will appear as the background images for Game Story and Achievement Prompts respectively. The Featured Image for Game Story will also appear as an icon for the Game on the Gamification subpage in View Mode.

  6. In the XP and Level tab, under Experience Points Settings, fill in the following detail:

    • Marks to XP Conversion - This converts marks awarded to students in standalone questions and quizzes into XP. The higher the Marks to XP conversion, the easier it is for students to level up. The default Marks to XP conversion is 1 mark to 50 XP.

    • XP Modifiers - There are three ways to differentiate XP received by students to encourage positive behaviour. These modifiers only affect the XP awarded, not the awarded marks on the Heatmap.

      • Early submission of Quizzes - The student receives a bonus for the entire Quiz if the Quiz is submitted early. The default bonus set for early submission is 50% and the default number of days to be considered an Early Submission is 1 day before the Section Due Date.

        Calculation: (1 + percentage⁄ 100) x Awarded Marks x Marks to XP Conversion

      • Late submission of Quizzes - The student receives reduced XP for the entire Quiz if the Quiz is submitted late. The default penalty set for late submission is -50%.

        Calculation: (1 - percentage⁄ 100) x Awarded Marks x Marks to XP Conversion

      • Retries for questions - Standalone questions and questions in Progressive Quiz can be given multiple tries. Students who get the questions right on their first try will be given the full XP while students who get the questions right on subsequent tries will get a reduced XP.

        Calculation: (1 - percentage⁄ 100) number of retries x Awarded Marks x Marks to XP Conversion

  7. In the XP and Level tab, under Level Progression Settings, fill in the following detail:

    • Base XP and XP Increment Per Level - This determines how XP to Next Level will be changed over different levels.

      • Base XP is the number of XP required for students to get from Level 1 (Base Level) to Level 2 (Next Level).

      • XP Increment Per Level is the increment in XP to Next Level from the previous level. This causes the XP required to Next Level to increase as the level of the student gets higher.

        Calculation: XP required to Next Level = Base XP + (Current Level - 1) x XP Increment per Level

      Note: The Level Summary table will be generated as changes are made in the two input fields.

  8. In the Game Story tab, click Add Game Story to create a Game Story. You will need to enter the following details:

    • Title - The game story title will be pre-filled for you as “Untitled Game Story”. Change the Title to one that is concise and unique to help you identify the Game Story.

    • Description - The description allows you to write the story.

    • Image - This image will appear as the featured image of the prompt when this Game Story appears for the student. If left empty, a default Game Story icon will be populated for you.

    Note: To determine when the Game Story will appear during the course, read Add Conditions for Game Stories and Achievements. All Game Stories are hidden on default until they are awarded to the students.

  9. In the Achievements tab, click Add Achievement to create an Achievement. You will need to enter the following details:

    • Title - The achievement title will be pre-filled for you as “Untitled Achievement”. Change the Title to one that is concise and unique to help you identify the Achievement.

    • Description - The description allows you to describe the achievement and write congratulatory messages along with the achievement.

    • Icon - This icon will appear as the featured image of the prompt when this Achievement appears for the student and as an icon for this Achievement on the Leaderboard. If left empty, a default Achievement icon will be populated for you.

  10. Select the checkbox beside Hide Achievement from student if you want the achievement to be hidden from students until it is awarded to them. When unchecked, the achievement will be shown to students. Students will not own the achievement until they fulfill the conditions.

  11. Click Save to proceed.

Note: You can edit the Game Experience from the Monitoring Assignment page. Click the Game Experience tab, followed by Edit Game Experience.


Add a Game Story or Achievement

  1. In the Gamification subpage, click Add Game Story/Achievement.

  2. Alternatively, click the Overflow (3-dot icon) on the Game Story/Achievement Card to Add Game Story/Achievement After/Before.


Edit Game Story or Achievement

  1. In the Gamification subpage, click Game Story/Achievement.

  2. Click Edit under the Game Story/Achievement Card. You will be able to edit the Title, Description and Image.

  3. Click Save to confirm your changes.


Edit Game Story or Achievement

  1. Click the Overflow (3-dot icon) on the Game Story/Achievement Card and select Delete.

  2. Click OK to confirm.
