Add New Activity to a Lesson/Course (Improved)

Note: Starting from R18, activities (including Quizzes) have the option of being optional. This can be done by checking the box beside Set Activity/Quiz as optional. Optional Activities/Quizzes are indicated with in the Activity List.

Teachers can create Team Activities in Quizzes for Free-Response Questions and Audio-Response Questions.

  1. In the Lesson or Course Editor, there are multiple ways to add an Activity:

    • Click the Expand Activity List icon to open the Full Activity List. Select + Activity.

    • Click the Expand Activity List icon to open the Full Activity List. Click the Overflow (3-dot icon) found to the right of an Activity and click Add Activity After/Before.

    • From the collapsed Activity List, click + and select Activity.

    • From the Lesson/Course Plan, click Add Activity.

  2. To create a new Activity, you can choose from the following options:

    • Custom Activity (Create an activity with stand-alone components without quiz monitoring functions)

    • Progressive Quiz (Students receive feedback automatically after submitting each question)

    • Auto-graded Quiz (Students receive feedback automatically after submitting the quiz)

    • Teacher-marked Quiz (Students receive feedback after teachers mark and release the quiz)

    • Activity Templates (Prepopulated Activities aligned to the Active Learning Processes). Click to find out how to use an Activity Template for an Activity.

  3. Edit the Activity Title to describe the Activity. This will be visible to students.

  4. Click Save and X to close the Lesson/Course Plan.

  5. If you have selected a Lesson/Section Template (e.g. Acquisition, Discussion) when you created your Lesson/Section, your Lesson/Section will be pre-populated with activities. These activities are tagged to Active Learning Processes. To understand more about the Activities, you can click and view the Activity Notes in the Activity Details Card found in the Settings Drawer.

  6. To edit the Activity Notes and Active Learning Process in the Activity DetailsCard, click Edit on the Activity Details Card or Edit Details from the Overflow (3-dot icon). These fields are optional and will not be visible to students. Click Save to proceed.
