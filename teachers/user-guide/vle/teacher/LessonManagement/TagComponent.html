Tag a Question

Most components can be tagged to Subject, Level, Content Map, Cognitive Level, Topics, and Sub-topics so as to improve searchability in Community Gallery, MOE Library, and My Drive.

To include a question in Learning Progress, you have to tag a question before selecting the Include in Learning Progress checkbox. Tagging a question will allow you to view the Learning Progress of students. Click here to find out more about Learning Progress.

Note: Text/Media, ITT and Poll cannot be included in Learning Progress.

  1. In Lesson/Course Editor, click Edit on the Component Card. Alternatively, you can click the Overflow (3-dot icon) and select Edit.

  2. To include the Question in the Learning Progress, check the box beside Include in Learning Progress. You can also add Keyword Tags and Question Tags.

    Note:

    • You will need to add a Question Tag in order to include the question in the Learning Progress.

    • You are encouraged to only add one Question Tag for each question.

  3. Click the dropdown arrow located on the right of the Question Tag field.

  4. You can choose to select topics from the teaching subjects assigned to you (Assigned Subjects), subjects you are following (Followed Subjects), or from any other subject in the MOE Content Map (Other Subjects).

  5. Select the Subject from the dropdown list.

  6. Select the Level from the dropdown list. This list is based on the Subject chosen.

  7. Select the Content Map (previously known as Syllabus).

  8. Select the Cognitive Level from the dropdown list.

  9. Select the checkboxes next to the Topics or Sub-topics that are relevant to this question. You can click the downward arrow to expand the topics in order to see the sub-topics.

  10. Click the upward arrow on the Lesson/Course Tag field to close the selection.


Add More Question Tags (Not Advised)

Note: You are not advised to add more than one Question Tag to a Question as it will affect the data analytics of Learning Progress.

  1. To add more Question Tags, click Add Question Tag.

  2. Multiple Question Tags will be displayed in an accordion format.

  3. To expand or collapse the tag details for viewing, click the Question Tag Title or the upward/downward arrow. To remove the tag, click the Trash icon.

  4. Click Save to exit.
