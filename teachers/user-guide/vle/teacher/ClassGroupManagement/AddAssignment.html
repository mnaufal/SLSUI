Add Assignment to Class Group (New)

  1. You can add assignments to the Class Group from the Class Group page.

  2. Access Class Groups from the left menu or by selecting the Class Group (if pinned) on your homepage. On the Class Group page, click Add Assignment to add the assignment:

    • From New Lesson or Course

    • From My Drive

    • From Library

    • From Starred Resources

  3. You may also add a new category to your Class Group assignment. This can be used to organise the assignments for your Class Group.
